# Claude Code Agentic Mode — Corso da 3 ore

Materiali per un corso pratico su Claude Code: subagent, hook, skill, scripting, workflow quotidiani e sicurezza.

## Contenuti
- `docs/AGENDA.md`: agenda da 180 minuti.
- `.claude/agents`: agenti specializzati.
- `.claude/hooks`: guardrail e audit.
- `.claude/skills`: skill riutilizzabile.
- `scripts`: automazioni con `claude -p`.
- `labs`: esercitazioni guidate.

## Prerequisiti
- Claude Code aggiornato.
- Git.
- Bash/Git Bash o WSL.
- Python 3 per gli hook demo.

## Avvertenza
Gli esempi sono didattici. Prima dell'uso aziendale adattare allowlist, denylist, logging, gestione segreti e policy di rete.
