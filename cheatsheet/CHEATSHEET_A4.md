# Claude Code Agent Engineering — Cheat Sheet

## Formula agente
`name + description + tool minimi + protocollo + output verificabile`

## Formula hook
`evento + matcher + comando deterministico + exit code chiaro`

## Formula workflow
`scope -> agenti -> ordine -> stop condition -> evidenze -> handoff`

## Regola sicurezza
Se può scrivere, cancellare, inviare o leggere segreti: chiedere conferma o bloccare.

## Prompt base
"Prima piano e rischi. Poi modifiche piccole. Cita evidenze. Non usare segreti. Non fare push."
