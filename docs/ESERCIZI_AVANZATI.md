# Esercizi avanzati

## Esercizio 1: pattern agentico 1

### Scenario
Hai una modifica software con rischio medio e devi usare Claude Code senza aumentare il rischio operativo.

### Compito
1. Scegli 2 agenti dal catalogo.
2. Limita i tool al minimo.
3. Definisci un hook di controllo.
4. Scrivi un prompt iniziale.
5. Definisci output e stop condition.

### Valutazione
- Il workflow è ripetibile?
- I rischi sono espliciti?
- Esiste un fallback manuale?
