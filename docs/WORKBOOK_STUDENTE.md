# Workbook studente

## Prima del corso
- Installa Claude Code.
- Clona o scarica questo repository.
- Verifica `git status` pulito.
- Crea un branch `lab/claude-code-agentic`.

## Durante il corso
Per ogni laboratorio compila:
- Obiettivo.
- Agente usato.
- Tool concessi.
- Rischi.
- Output prodotto.
- Miglioramento possibile.

## Dopo il corso
Scegli un'attività ripetitiva del tuo lavoro quotidiano e trasformala in:
1. agent;
2. hook;
3. workflow;
4. checklist sicurezza.
